# Sistema Taiji

Commercial web product for Chinese medicine clinics and solo practitioners. Handles patient management, scheduling, intake forms (anamnese), and clinic operations. Being built as a SaaS.

## Stack

Next.js (App Router) + Supabase (auth + database) + TypeScript + Tailwind + shadcn/ui. Deployed on Vercel.

## Departments

- **R&D/** — product development: code, agents, testing, deployment.
- **Marketing/** — acquisition, content, positioning.

More departments (Support, Finance, etc.) to be added as the business grows.
